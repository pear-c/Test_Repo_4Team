# 댓글 관리 기능 설계

1. 댓글 작성
   - 사용자로부터 댓글 내용을 입력받는다.
   - 댓글에 포함되는 정보:
     - 댓글 ID: 고유한 번호로 지정.
     - 작성자 이름: 댓글 작성자의 이름.
     - 댓글 내용: 사용자가 입력한 텍스트.
     - 작성 시간: 댓글 작성 시간을 자동으로 기록.
   - 댓글 데이터를 저장소(리스트, 파일, DB 등)에 저장.

2. 댓글 조회
   - 특정 상품에 달린 댓글 목록을 가져온다.
   - 반환되는 댓글 정보:
     - 댓글 ID, 작성자 이름, 댓글 내용, 작성 시간.
   - 최신 댓글이 먼저 표시되도록 정렬.

3. 댓글 삭제
   - 관리자만 댓글을 삭제할 수 있다.
   - 삭제할 댓글 ID를 입력받아 해당 댓글을 삭제.
   - 댓글 ID가 유효하지 않을 경우 오류 메시지 반환.
